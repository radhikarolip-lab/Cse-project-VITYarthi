# Student Management System 

This project is a simple and beginner-friendly *Student Management System* developed using Python.  
It helps store and manage student records such as name, roll number, branch, and marks.

# Features

Add Student Record  
View All Student Records  
Search Student by Roll Number  
Update Existing Record  
Delete Record  
Data stored permanently using JSON file  
Easy menu-based system  

# Technologies Used

- Python
- JSON File Handling
- Modular Programming
